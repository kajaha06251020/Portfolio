# 状態異常システム仕様書

## 概要
FFアニバーサリー「Final Fantasy X-99」の状態異常システム実装。
複数のステータス異常タイプとその相互作用を管理。

## システム構成

### 1. 状態異常の定義

#### ステータス異常タイプ（8種類）

| 異常名 | ID | 説明 | 効果 | 解除方法 |
|--------|-----|------|------|--------|
| 毒 | `poison` | 毒状態。毎ターン小ダメージ | 毎ターンダメージ（最大HP × 5%） | 解毒薬、ケアル系 |
| 睡眠 | `sleep` | 睡眠状態。行動不可 | 行動選択不可。物理攻撃で解除 | 目覚まし、物理攻撃 |
| 麻痺 | `paralysis` | 麻痺状態。行動順遅延 | ATB進行速度-50% | 全快系、麻痺治療 |
| 暗闇 | `blind` | 暗闇状態。命中率低下 | 物理/魔法命中-25% | 目薬、全快系 |
| 混乱 | `confuse` | 混乱状態。ランダム行動 | 敵/味方ランダムに行動 | 静寂、全快系 |
| 石化 | `petrify` | 石化状態。完全行動不可 | 行動完全不可。段階的に進行 | 白魔法プリス、全快系 |
| スロウ | `slow` | 減速状態。速度低下 | 速度-50% | ヘイスト、全快系 |
| ストップ | `stop` | 停止状態。行動完全不可 | ATB凍結。時間経過で解除 | 期間満了、ハイポーション系 |

### 2. 状態異常の管理

#### 状態異常データ構造
```json
{
  "status_effects": {
    "poison": {
      "name": "毒",

      "icon": "poison.png",
      "severity": 2,
      "resistance_type": "poison_resistance",
      "can_stack": false,
      "duration": -1,
      "effect_function": "apply_poison_damage"
    },
    ...
  }
}
```

```json
{

    {
      "effect_id": "poison",

      "source": "enemy"
    },
    {
      "effect_id": "sleep",

      "duration": 2,
      "source": "enemy"
    }
  ]
}
```

### 3. 状態異常の適用ルール

#### 適用判定
```
適用判定 = 1 - 抵抗值（0～1）
例：毒抵抗0.3 → 70%の確率で適用
```

#### 継続時間
| 異常 | 継続期間 | 説明 |
|------|--------|------|
| 毒 | 無期限 | 解除まで続く |
| 睡眠 | 物理攻撃で解除 | または1～3ターン |
| 麻痺 | 3ターン | または手動解除 |
| 暗闇 | 戦闘終了まで | または解除アイテム |
| 混乱 | 2～4ターン | ランダムターン |
| 石化 | 段階進行 | 3段階で完全化 |
| スロウ | 無期限 | ヘイストで解除 |
| ストップ | 1～2ターン | 時間経過で自動解除 |

### 4. 状態異常の効果処理

#### 毒（poison）
- **毎ターン処理**: HP減少（最大HP × 5%）→ ダメージポップ表示
- **視覚効果**: 画面フラッシュ（薄紫）
- **回避不可**: 防御状態でも効果は続く

#### 睡眠（sleep）
- **バトルロジック**: アクター選択不可状態
- **解除条件**: 
  - 物理攻撃を受けた時点で自動解除
  - または `目覚まし` アイテム使用
- **優先度**: 混乱より高優先度（睡眠中は混乱判定なし）

#### 麻痺（paralysis）
- **ATB影響**: ATB進行速度が50%に減速
- **計算式**: `new_atb_increment = base_increment × 0.5`
- **視覚効果**: キャラ画像に青い電撃エフェクト

#### 暗闇（blind）
- **命中率**: 物理魔法両方-25%
- **計算式**: `adjusted_accuracy = accuracy × 0.75`
- **攻撃側判定**: ダメージ型アクションの最後に判定

#### 混乱（confuse）
- **行動選択**: ランダム敵に行動
  - 50%: 敵を攻撃
  - 30%: 味方を攻撃
  - 20%: 何もしない
- **対象選択**: 完全ランダム
- **スキップ不可**: メニュー選択をスキップ

#### 石化（petrify）
- **段階制**:
  - Lv1: 回避率-50%、行動速度-30%
  - Lv2: 行動選択不可、防御のみ可能
  - Lv3: 完全石化（行動完全不可、ダメージ無効化）- **解除方法**: `プリス` 白魔法のみ可

#### スロウ（slow）
- **ATB影響**: ATB進行速度が50%に減速（麻痺と同じ）
- **異なり**: 魔法系の減速（属性耐性の対象）
- **適用順序**: 複数同時は効果を重ねない

#### ストップ（stop）
- **ATB凍結**: 完全に停止状態
- **回避不可**: 防御状態でも停止
- **自動解除**: 期間満了で自動復帰
- **表現**: キャラが静止画のままになる

### 5. 複合効果と優先度

#### 複数異常同時適用時の優先度

1. **ストップ**: 最高優先度（全行動不可）
2. **石化Lv3**: 行動完全不可
3. **睡眠**: 行動選択不可
4. **混乱**: ランダム行動
5. **その他**: 能力値変化のみ

#### 互いに打ち消す異常
```
スロウ + ヘイスト → スロウ解除（またはヘイストが優先）
毒 + 全快 → 毒解除
```

### 6. 実装モジュール

#### `src/battle/status_effects.py`
- `StatusEffect` クラス：状態異常定義
- `StatusEffectManager` クラス：状態異常管理
- 効果処理関数群（apply_poison_damage等）

#### 公開API
```python
status_mgr = StatusEffectManager()

# 状態異常適用
status_mgr.apply_effect(actor, effect_id, duration)

# 状態異常確認
has_effect = status_mgr.has_effect(actor, effect_id)
effects = status_mgr.get_active_effects(actor)

# 状態異常解除
status_mgr.remove_effect(actor, effect_id)

# 毎ターン処理
status_mgr.on_turn_start(actor)
```

### 8. ゲームバランス調整

#### 状態異常の発生確率
- **敵の魔法**: 60～80%で状態異常を付与
- **プレイヤー側**: アイテム/魔法で状態異常を回復

#### 状態異常への耐性
```
抵抗値 = 基本耐性 + ジョブ補正 + 装備補正
例: 麻痺抵抗
  基本: 0.1 (10%)
  knight: +0.2 → 30%
  鎧の追加: +0.1 → 40%
```
